🔐 1. Oracle Database Unsupported Version
Plugin ID: 55786

CVSS v3 Score: 10.0 (Critical)

🔐 2. Oracle TNS Listener Remote Poisoning

Plugin ID: 69552

CVSS v3 Score: 7.3 (High)

Impact: Allows attackers to redirect database traffic

🔐 3. SSL Self-Signed / Untrusted Certificates

Plugin IDs: 51192, 57582, 15901

Impact: Users may be vulnerable to spoofing or MITM attacks
